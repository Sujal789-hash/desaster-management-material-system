# Disaster Management Material System

A web-based system to manage disaster relief materials efficiently.

## Features

* Material inventory management
* Track movement of materials
* Delivery and receipt tracking
* User management
* Dashboard for monitoring resources

## Technologies Used

* Node.js
* Express.js
* HTML
* CSS
* JavaScript

## Project Structure

```
public/   → Frontend files  
src/      → Backend logic  
server.js → Main server file
```

## Installation

1. Clone the repository

```
git clone https://github.com/Sujal789-hash/desaster-management-material-system.git
```

2. Install dependencies

```
npm install
```

3. Run the server

```
node server.js
```
